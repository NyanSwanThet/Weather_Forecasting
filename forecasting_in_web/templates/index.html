Weather Forecast

Project Description

We offer comprehensive weather information for the current day, including temperature forecasts updated on an hourly basis for a 5-day period. Additionally, our system predicts the precipitation conditions for tomorrow, indicating whether it will be rain-free, experience light rain, moderate rain, or heavy rain.

Our forecasting system utilizes a deep neural network that has been trained on five years of historical weather data for Mandalay, spanning from 2018 to 2022. The prediction system receives real-time weather data via an API as input.
